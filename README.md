## Deskripsi Proyek
Proyek ini merupakan dashboard analisis data interaktif yang dikembangkan menggunakan Streamlit. Dashboard ini bertujuan untuk menganalisis pola penyewaan sepeda berdasarkan data historis, dengan fokus pada pengaruh faktor cuaca, musim, jenis hari (hari kerja dan hari libur), serta jam operasional.

## Setup Environment - Anaconda
```bash
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Run steamlit app
```bash
streamlit run dashboard.py
```
